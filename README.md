# onetalk-web
Repositório destinado ao front do OneTalk, que é uma plataforma de vídeo chamadas 1:1, construída no intuito de estudo pessoal, com autenticação JWT, gerenciamento de amigos e chamadas em tempo real usando WebRTC. Desenvolvido com Quarkus, MongoDB e WebSockets. Foco em segurança e simplicidade para comunicação entre usuários.
